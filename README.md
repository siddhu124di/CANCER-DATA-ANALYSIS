# Cancer Data Analysis Project

## Project Overview

This project performs an end-to-end analysis of a cancer patient dataset containing approximately 50,000 records. The objective was to identify factors affecting cancer severity, survival outcomes, and treatment-related characteristics using statistical analysis and machine learning techniques.

---

## Objectives

* Perform data cleaning and preprocessing
* Explore relationships between patient attributes and cancer severity
* Analyze survival patterns across different cancer stages
* Apply statistical hypothesis testing
* Build predictive machine learning models
* Extract actionable insights from healthcare data

---

## Key Findings

* Smoking and Genetic Risk showed strong relationships with cancer severity.
* Cancer stages exhibited different survival characteristics.
* Interaction analysis indicated that Genetic Risk did not significantly modify the effect of Smoking on cancer severity.
* Several demographic and medical factors contributed to prediction performance.
